# 会话索引模板

> 本模板用于规范会话索引的格式，确保每次生成的索引内容完整、结构清晰。

---

## 基本信息

```markdown
## 基本信息
- 时间：YYYY-MM-DD
- 会话ID：当前会话 / 具体会话ID
- 持续时间：约 XX 分钟
- 上下文使用：XX%
```

## 主题和关键词

```markdown
## 主题
[一句话描述本次会话的核心主题]

## 关键词
[关键词1]、[关键词2]、[关键词3]...
```

## 摘要

```markdown
## 摘要
[一句话总结本次会话完成了什么、有什么成果]
```

## 详细内容

### 任务完成情况

```markdown
### 任务完成情况

| 任务 | 状态 | 交付物 | 核心内容 |
|------|------|--------|---------|
| 任务1 | ✅/⏳/❌ | 文件名.md | 简要描述 |
| 任务2 | ✅/⏳/❌ | 文件名.md | 简要描述 |
```

### 关键决策

```markdown
### 关键决策

| 决策 | 内容 | 原因 |
|------|------|------|
| 决策1 | 具体内容 | 为什么这么做 |
| 决策2 | 具体内容 | 为什么这么做 |
```

### 踩坑记录

```markdown
### 踩坑记录

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 问题1 | 为什么会出现 | 怎么解决的 |
| 问题2 | 为什么会出现 | 怎么解决的 |
```

### 用户偏好更新

```markdown
### 用户偏好更新

| 偏好类型 | 新增内容 |
|---------|---------|
| 分析偏好 | 具体内容 |
| 执行偏好 | 具体内容 |
```

### 待办

```markdown
### 待办

- [ ] 待办事项1
- [ ] 待办事项2
```

## 参考

```markdown
## 参考

- 生成文件：文件1.md、文件2.md
- 相关链接：URL
```

---

## 使用说明

1. **实时生成**：每次会话结束时，按此模板生成索引
2. **离线生成**：每周执行脚本时，按此模板生成索引
3. **格式要求**：
   - 使用 markdown 格式
   - 文件名：`YYYY-MM-DD.md`（实时）或 `YYYY-MM-DD_会话ID前8位.md`（离线）
   - 内容完整：基本信息 + 主题关键词 + 摘要 + 详细内容 + 参考
4. **验证**：生成后检查文件是否存在，确保不遗漏
