## 元数据规范

每个文件必须包含以下元数据：

```yaml
---
title: [主题名称]
category: [所属一级分类]
difficulty: [⭐ / ⭐⭐ / ⭐⭐⭐]
source: [原创 / 整理 / 转载]
status: [草稿 / 完善 / 定稿]
created: [YYYY-MM-DD]
updated: [YYYY-MM-DD]
tags: [标签1, 标签2]
related: [相关主题链接]
---
```

## 文件命名规范

```
[编号].[主题名称].md

示例：
01.闭包.md
02.事件循环.md
01.1.手写Promise.md（子主题用.1编号）
```
